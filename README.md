# Iris Flower Classification using ANN

## Project Overview
This project uses an Artificial Neural Network (ANN) built with TensorFlow/Keras to classify Iris flowers into:
- Setosa
- Versicolor
- Virginica

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- TensorFlow/Keras

## Workflow
1. Data Loading
2. Data Preprocessing
3. Feature Scaling
4. Label Encoding & One-Hot Encoding
5. Train-Test Split
6. ANN Model Building
7. Model Training
8. Evaluation
9. Model Saving
10. Prediction on Unseen Data

## Results
The model achieved high classification accuracy on the Iris dataset.

## Conclusion

The Artificial Neural Network (ANN) model was successfully developed and trained to classify Iris flowers into Setosa, Versicolor, and Virginica. After preprocessing, feature scaling, and model training, the ANN achieved high classification accuracy, demonstrating its effectiveness for multiclass classification tasks. The trained model accurately predicted unseen flower samples and was successfully saved for future use
